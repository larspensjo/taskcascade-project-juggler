# Visual Design — Dark Theme

TaskCascade uses a calm, dense workspace rather than a marketing surface. The shell is low-contrast, while active information and the single blue accent guide attention.

- Canvas: `#0a0b0d`; panels: `#0e1114` and `#14181d`; hairlines: `#20242b`.
- Primary text: `#f5f6f7`; secondary text: `#a8acb3`; muted text: `#6b7280`.
- Accent: `#0052ff` for primary actions and selection; use `#4d8bff` for linked text on dark backgrounds.
- Completion is `#16c784`, applied sparingly as an icon/text signal.
- Use Inter for prose/UI and JetBrains Mono for ordering/counts. Panels use hairlines and surface steps instead of shadows; overlays are the exception.
- Keep actions compact and keyboard discoverable. Primary buttons are pills, list rows are rounded but dense, and task ordering remains visually prominent.
